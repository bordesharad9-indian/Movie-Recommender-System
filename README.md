Movie Recommender System

A content-based "Movie Recommendation System" built using "Python", "Streamlit", and "Machine Learning", which recommends similar movies based on user selection .

---

Features
- Select a movie from dropdown
- Get top 5 similar movie recommendations
- Fetch movie posters using TMDB API
- Interactive web interface using Streamlit
- Fast and lightweight ML model using cosine similarity



  Tech Stack
- Python 
- Streamlit
-Pandas
-Scikit-learn
-Requests
-Pickle
